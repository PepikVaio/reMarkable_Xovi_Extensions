# Screen mode!
Your reMarkable? Your screen settings.


## Description
Don't be tied to reMarkable and grab the screen update at your discretion..

  - Option to set global range.
  - Option to activate quick settings.
  - Option to set different screen modes.
  - Option to set color refresh pause.

  - Next will soon ...
